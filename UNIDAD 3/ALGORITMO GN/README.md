# Proyecto: Optimización con Algoritmo Genético (GN)

## Descripción

Este proyecto se enfoca en la aplicación de **Algoritmos Genéticos**, una técnica de optimización bioinspirada, para resolver un problema complejo. El objetivo es encontrar soluciones óptimas o casi óptimas mediante la simulación de procesos de evolución natural como la selección, el cruce y la mutación. El problema a resolver es el problema del agente viajero **TSP**.
