# PROYECTO DETECCIÓN DE PLACAS
Dentro de la carpeta de UNIDAD 4, se encuentran las tres carpetas que conforman nuestro proyecto de detacción de placas vehiculares,  

## Carpeta API: 
- **Descripción:** En la carpeta API se encuentra toda la lógica de nuestro backend para la comunicación cliente servidor, utilizando python para realizar la fastapi. 

## Carpeta DetectorPlacas: 
- **Descripción:** En esta carpeta se encuentra la lógica de nuestro frontend utilizando React Native con Expo para el desarrollo de la aplicación móvil.

## Carpeta ModeloCNN:
- **Descripción:** Dentro de la carpeta ModeloCNN se encuentra la lógica del proceso de entrenamiento de nuestro modelo de visión artificial utilizando YOLO.
